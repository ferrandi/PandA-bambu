* MIG network: ``mockturtle/networks/mig.hpp``
* XAG network: ``mockturtle/networks/xag.hpp``
* XMG network: ``mockturtle/networks/xmg.hpp``
* *k*-LUT network: ``mockturtle/networks/klut.hpp``
